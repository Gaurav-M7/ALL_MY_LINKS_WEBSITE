# GAURAV MAPARI // ARCHETYPE
A premium, highly interactive personal link-in-bio website inspired by anime character GFX (Graphic Effects) and edit poster aesthetics. Featuring brutalist typography, dynamic CRT effects, scanline grids, and responsive split-screen layouts.

LIVE PREVIEW OF WEBSITE :https://gauravmapari.netlify.app/
---
## 🎨 Visual Aesthetics & Design
This project departs from traditional minimal link-in-bio setups to offer a unique visual experience:
- **GFX Edit Poster Style:** High contrast, custom grid layouts, and industrial framing.
- **Micro-Animations:** Interactive link hover offsets, dynamic floating coordinates, and custom cursor elements.
- **Cyberpunk Accents:** Active scanlines, subtle digital noise, neon highlight styling, and a running UTC clock.
- **Theme Palette:** Deep off-blacks, grid-gray structures, and stark neon-red accents.
## 🛠️ Tech Stack
Built pure and lightweight for optimal performance and rapid loading:
* **HTML5:** Semantic architecture.
* **Vanilla CSS3:** Clean, layout-driven styling using modern CSS Variables, Flexbox, Grid, keyframe animations, and filter effects.
* **Vanilla JavaScript:** Responsive interactive components and system clock features.
## 🚀 Getting Started
To view the project locally:
1. **Clone the Repository:**
   ```bash
   git clone https://github.com/Gaurav-M7/linktree.git
   cd linktree
   ```
2. **Run Locally:**
   Simply open `index.html` in your browser, or spin up a local development server:
   ```bash
   # Using VS Code Live Server extension, or:
   npx serve .
   ```
